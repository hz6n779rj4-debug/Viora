# KASNEST Free Deployment Guide

## Best free hosting choice
Use **Cloudflare Pages** for the web app.

Why:
- free custom domain support
- global CDN
- good fit for static web builds
- simple GitHub deploy flow

## Build command
```bash
yarn install
yarn build-web
```

## Output directory
The web export is generated by Expo. After build, deploy the generated web output directory created by the project build script.

## Recommended setup
### Web
- Host: Cloudflare Pages
- Repo: GitHub
- Framework preset: None / static site
- Build command: `yarn build-web`

### Mobile preview builds
- Use Expo / EAS for Android and iOS test builds
- Good for testing without managing your own Mac build pipeline

## Important
This codebase is still based on the original AT Protocol / Bluesky client stack, so to turn it into a fully custom KASNEST network you will still need backend and auth customization.
