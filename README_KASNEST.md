# KASNEST fork

This package is a KASNEST-branded starter built from the uploaded Bluesky-based social client.

What was changed:
- App name, slug, scheme, and primary brand color
- KASNEST logo, wordmark, favicon, and app icon
- Branded mobile and web splash screens
- Browser/page title and a few visible labels

Important:
- This is still based on the original AT Protocol / Bluesky client stack.
- Many deeper backend references and account/server flows still point to the original ecosystem unless you continue the backend rework.

Run locally:
1. yarn
2. yarn web

For native builds, configure your own bundle IDs, associated domains, and backend env variables before shipping.
